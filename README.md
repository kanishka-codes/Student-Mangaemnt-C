# Student Management System in C

## Description
This is a C program to manage student records. 
It allows you to enter student details, calculate totals, averages, grades, and save results to a file.

## Features
- Enter student details (Roll, Name, Marks)
- Calculate total, average, and grade
- Display a result table
- Show the topper
- Save all data to a file (`students.txt`)
